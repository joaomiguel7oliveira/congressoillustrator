Nota

Para criar um padrão decorativo (seamless pattern) de forma rápida no Illustrator: Desenhe o elemento que será a base da estampa. Arraste esse desenho para dentro do painel Amostras (Swatches). Selecione qualquer forma nova (ex: um retângulo grande) e clique na amostra do padrão que você acabou de criar para preenchê-la. Para edições avançadas de encaixe, vá em Objeto > Padrão > Criar.
